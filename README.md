# Bootcamp Box

Bootcamp Box is an all-in-one, AI-powered coding adventure designed for beginners. Each self-contained section blends hands-on coding challenges, interactive discussions, and personalized projects. This course spans the entire computer science bachelor’s-level curriculum.

Built around the philosophy of learn by doing, Bootcamp Box keeps learners active, curious, and creative from day one. Every lesson is guided, evaluated, and adapted by Claude Code, your personal AI instructor — teaching, reviewing, and collaborating with you like a real mentor on call 24/7.
## Quick Start

1. **Download**: [Click here to download](https://github.com/pixelpax/bootcamp-box/archive/main.zip) and unzip

2. **Open Terminal**:
   - **Mac**: Press `Cmd+Space`, type "Terminal", press Enter
   - **Windows**: You'll need WSL (Windows Subsystem for Linux). [Install WSL](https://learn.microsoft.com/en-us/windows/wsl/install), then open Ubuntu from the Start menu
   - **Linux**: Open your terminal app

3. **Navigate to the folder**:
   ```
   cd ~/Downloads/bootcamp-box-main
   ```
   (Or wherever you unzipped it)

4. **Start Bootcamp Box**:
   ```
   bash bb
   ```
   This will install Claude Code if needed and start your first lesson.

## What You'll Need
- A computer (Mac, Windows, or Linux)
- [Install WSL](https://www.youtube.com/watch?v=wz0QBNy9i7w) if you're using Windows
- An internet connection
- That's it! The setup script handles the rest.

## Commands

Once you're in a session:
- `/submit` - Submit your work for grading (moves to next lesson on pass)
